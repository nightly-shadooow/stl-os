JavaScript must be enabled to use track.easterbunny.cc!

Hi! To use track.easterbunny.cc, JavaScript must be enabled. Please enable JavaScript in your browser settings, then restart your browser.
When you've enabled JavaScript, click here to return to the tracker.

Please note: If you're using iOS and are using the tracker from a homescreen shortcut, you'll need to clear website data to use the tracker after enabling JavaScript. Do this by going into Settings, scroll down to Safari, scroll down to Advanced, then tap Website Data, then scroll to the bottom and click Remove All Website Data. This will remove your personalized tracker settings. After that, clear "Easter Bunny Tracker" from your app switcher, then relaunch the tracker.
